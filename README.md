# MachineLearningUsingPython_Project
This is a Network Intrusion Detection System using ML and Python Project

◦ Developed a NIDS model using Python and machine learning techniques to identify potential security threats.
◦ Implemented both misuse and anomaly detection approaches to detect known and unknown attack patterns.
◦ Utilized the KDD dataset for training and evaluating models for threat classification.
◦ Applied ML Algorithms including SVM, LR, Random Forest, K-Nearest Neighbors, Decision Tree, and AdaBoost.
◦ Designed a system to continuously monitor network packets and compare real-time data against expected patterns for anomaly detection.
◦ Integrated packet analysis using a packet sniffer and decoder to detect malicious activities such as malware and trojans.
◦ Built logic for real-time alert generation to notify administrators upon detection of suspicious activities.
◦ Gained strong experience in data processing, pattern recognition, and algorithm-based problem solving.
